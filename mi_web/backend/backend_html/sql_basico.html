Sistema de Base de Datos Médica

Consultas SQL Básicas - Práctica Estudiantil

Instrucciones

Haz clic en cualquier botón para ver y ejecutar consultas SQL básicas. Estas consultas te permiten realizar operaciones fundamentales en la base de datos médica.

Nota: Si es la primera vez que usas el sistema, haz clic en "Cargar Datos" para cargar datos de ejemplo.

Operaciones SQL Básicas

SELECT - Consultar Pacientes

INSERT - Insertar Nuevo Paciente

UPDATE - Actualizar Información del Paciente

DELETE - Eliminar Registros

CREATE TABLE - Crear Nueva Tabla

DROP TABLE - Eliminar Tabla Completa
